# Java 程式作業 - 模擬販賣機

## 作業說明
請同學實作一個模擬販賣機的程式，需透過 **console 介面** 提供操作選項。程式必須包含以下功能：

### 功能需求
1. **操作選單**
   - 投幣
   - 退幣
   - 購買
   - 離開程式

2. **投幣 (30%)**
   - 要求使用者輸入投幣金額
   - 投幣後回到操作選單

3. **退幣 (15%)**
   - 將目前投入的金額歸零
   - 回到操作選單

4. **購買 (50%)**
   - 要求使用者選擇欲購買的飲料
   - 飲料種類：
     - 礦泉水：15 元
     - 奶茶：20 元
     - 可樂：25 元
   - 若金額足夠：
     - 扣除購買金額
     - 顯示餘額與購買的飲料名稱
   - 若金額不足：
     - 顯示提示並回到操作選單

5. **離開程式 (5%)**
   - 結束程式

### 輸入檢查
可假設助教為正常使用者，但仍建議加入非法輸入檢查，以避免程式錯誤。

---

## 繳交方式
- 請將 **程式碼複製到 `.txt` 檔**，再上傳以方便批改。
- 程式需以 `Java` 撰寫，並可在命令列環境下正常執行。

---

## 評分比例
- 投幣功能：30%
- 退幣功能：15%
- 購買功能：
  - 顯示飲料種類與價格：15%
  - 購買成功後餘額與名稱顯示：30%
  - 錢不夠時回到操作選單：5%
- 離開程式：5%
